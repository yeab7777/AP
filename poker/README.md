# 🎲 Poker Game (Advanced Programming Project)

A Java-based implementation of the classic **Texas Hold’em Poker** game, built as part of advanced programming coursework. This project demonstrates object-oriented design, modular architecture, and game logic implementation.

---

## 📌 Features

- **Deck & Card System**
  - 52-card deck with shuffle and deal functionality.
  - Object-oriented classes for `Card`, `Deck`, and `Player`.

- **Game Flow**
  - Texas Hold’em sequence: Pre-flop → Flop → Turn → River → Showdown.
  - Community cards dealt in correct order.

- **Betting System**
  - Player actions: fold, check, call, raise.
  - Pot management and chip tracking.

- **Hand Evaluation**
  - Ranking system from Royal Flush → High Card.
  - Showdown logic to determine the winner.

- **Extendable Design**
  - Ready for GUI integration with JavaFX.
  - Can be expanded for AI opponents or online multiplayer.

---

### Requirements
- Java 17+
- IntelliJ IDEA or Command Prompt
- (Optional) JavaFX SDK for GUI
