# 📝 Notepad App (Advanced Programming Project)

A JavaFX-based **Notepad application** with database integration via MySQL (XAMPP).  
This project demonstrates GUI design, modular coding, and persistence of notes with formatting options.

---

## 📌 Features

- **Text Editing**
  - Create, edit, and save notes.
  - Font selection (Arial, Times New Roman, Courier New).
  - Highlighting and bold formatting.

- **Bookmarks**
  - Mark important notes for quick access.
  - Stored in the database for persistence.

- **Database Integration**
  - Notes saved in MySQL using JDBC.
  - Supports title, content, and bookmark status.

- **User-Friendly Layout**
  - Menu bar for file and format options.
  - Clean JavaFX interface with modular design.

---

### Requirements
- Java 17+
- JavaFX SDK
- MySQL Connector/J
- XAMPP (for database hosting)
